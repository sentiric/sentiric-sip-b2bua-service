# 🔄 Sentiric Sip B2Bua Service

SIP B2BUA olarak inbound ve outbound SIP oturumlarını yöneten Telekom çekirdeğidir.

## 🚀 Hızlı Başlangıç
```bash
cargo run --release
```

## 🏛️ Mimari ve Mantık
* **Geliştirici Kuralları:** Gizli [.context.md](.context.md) dosyasını okuyun (AI Ajanları için zorunludur).
* **Anayasal Konum:** [sentiric-spec/spec/services/sip-b2bua.spec.yaml](https://github.com/sentiric/sentiric-spec)
